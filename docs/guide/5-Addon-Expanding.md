# 💎 5. Các Addon mở rộng tính năng

## I. Addon tìm kiếm phụ đề
>* Addon [**SubMaker**](https://stremio-addons.net/addons/submaker-elfhosted)
>1. Bấm **<mark2>Install -> Configure</mark2>**
>2. Bấm **<mark2>Got it -> Quick Setup -> Just Fetch Subtitles</mark2>**
>3. Bật và thêm API key **<mark2>Subsource và Wyzie Subs.</mark2>** Phần Wyzie chỉ cần chọn **<mark2>Subf2m và SubDL</mark2>**
>4. Chọn **<mark2>Vietnamese (VIE)</mark2>**
>5. Bấm **<mark2>Save and Install</mark2>** -> Copy link và dán vào phần Addon trong Stremio
